## 三、高阶函数
- 函数作为参数传递

    - Array.prototype.sort
    
        接受一个函数当作参数：目的是对数组进行排序，这是不变的部分;而使 用什么规则去排序，则是可变的部分
    
- 函数作为返回值输出

- 高阶函数的应用场景：
    - 函数柯里化：
    

    // 普通的add函数
    function add(x, y) {
        return x + y
    }
    
    // Currying后
    function curryingAdd(x) {
        return function (y) {
            return x + y
        }
    }
    
    add(1, 2)           // 3
    curryingAdd(1)(2)   // 3
    
    好处：
    - 参数复用
    - 提前确认，只需求判断一次
