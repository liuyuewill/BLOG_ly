- 高阶函数：接收函数作为参数的函数，都可以叫做高阶函数
- 高阶函数的一种特殊用法：柯里化

### 柯里化

- 柯里化函数的运行过程：收集参数，最终在最里层的函数里执行

- ``` JS
  function add(a, b, c) {
      return a + b + c;
  }
  
  把add函数柯里化：
  function _add(a) {
      return function(b) {
          return function(c) {
              return a + b + c;
          }
      }
  }
  
  所以下面的运算等价：
  add(1, 2, 3);
  _add(1)(2)(3);
  ```
